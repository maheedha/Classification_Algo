STEPS TO RUN THE random_forest.py code

1) Place the dataset in the same folder that has the python code file random_forest.py
2) In the line no 269, input the name of the data set and assign it to the variable "inputfile"
3) In the line no 267, input the value of number of trees, and assign it to the variable "T"
4) In the line no 268, input the value of number of features to be selected, and assign it to the variable "m"
5) Run the code in the any of the IDE or from console. Performance metrics will be displayed in the console.
