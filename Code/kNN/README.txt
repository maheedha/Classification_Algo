STEPS TO RUN THE knn_multiple.py code

1) Place the dataset in the same folder that has the python code file knn_multiple.py
2) In the line no 135, input the name of the data set and assign it to the variable "inputfile"
3) In the line no 138, input the value of K, and assign it to the variable "k"
4) Run the code in the any of the IDE or from console. Performance metrics will be displayed in the console.
